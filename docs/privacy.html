Privacy Policy

Last updated: May 3, 2026

This privacy policy applies to the Chrome extension 豆瓣我关注的阅读动态显示短评 ("the extension").

Data collected

The extension does not collect, transmit, or store any personally identifiable information. No analytics, tracking, or telemetry of any kind is performed.

Local storage

The extension uses chrome.storage.local to cache book comment data on your own device. The cache stores a mapping of Douban book subject IDs to comment text fetched from book.douban.com. This data never leaves your device and is used solely to avoid redundant network requests on repeat visits.

Network requests

When you visit book.douban.com/updates, the extension fetches the public comment page for each book displayed (book.douban.com/subject/{id}/comments?sort=follows) using your existing Douban login session. These requests go directly to Douban's servers. No data is routed through any third-party server operated by this extension.

Permissions

  • storage — used to cache comment data locally on your device to minimise repeat network requests.
  • host permission: https://book.douban.com/* — required to fetch friend-sorted comment pages from Douban with your login session so that personalised results are returned.

Third parties

No data is shared with, sold to, or processed by any third party. The extension communicates exclusively with book.douban.com.

Contact

For questions about this policy, please open an issue on the GitHub repository.

隐私政策

最后更新：2026年5月3日

本隐私政策适用于 Chrome 扩展程序 豆瓣我关注的阅读动态显示短评（以下简称"本扩展"）。

数据收集

本扩展不收集、传输或存储任何个人身份信息，不进行任何形式的数据分析、追踪或遥测。

本地存储

本扩展使用 chrome.storage.local 在您的设备本地缓存书籍短评数据。 缓存内容为豆瓣书籍 ID 与对应短评文本的映射关系，该数据不会离开您的设备， 仅用于避免重复访问时的冗余网络请求。

网络请求

当您访问 book.douban.com/updates 时，本扩展会使用您已有的豆瓣登录状态， 为页面上显示的每本书请求其评论页面 （book.douban.com/subject/{id}/comments?sort=follows）。 所有请求直接发往豆瓣服务器，不经过本扩展的任何第三方服务器中转。

权限说明

  • storage — 用于在本地设备缓存短评数据，减少重复网络请求。
  • host 权限：https://book.douban.com/* — 需要以用户登录状态请求豆瓣按好友排序的评论接口，以获取个性化的好友评论数据。

第三方数据共享

本扩展不向任何第三方共享、出售或提供数据处理服务。扩展仅与 book.douban.com 通信。

联系方式

如对本政策有任何疑问，请在 GitHub 仓库 提交 issue。
